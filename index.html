Einmalige Migration aus Google Sheets

Übernimmt alle Kategorien, Wochen und Werte aus dem bisherigen Google Sheet in dein Cloud-Konto. Vorher oben anmelden. Bereits vorhandene Einträge werden dabei überschrieben (gleiche Woche + Kategorie).

Einrichtung (einmalig pro App-Kopie)

  1. Auf supabase.com ein kostenloses Projekt anlegen (Region: Frankfurt).
  2. Im Dashboard: SQL Editor → New query, den Inhalt von supabase-setup.sql einfügen und ausführen.
  3. Unter Settings → API die Project URL und den anon public Key kopieren und oben in app.js bei SUPABASE_URL / SUPABASE_ANON_KEY eintragen.
  4. Hier registrieren & anmelden – fertig. Weitere Nutzer derselben App-Kopie müssen sich nur registrieren (Schritte 1–3 entfallen).
