Google-Sheets-Verbindung

Einrichtung in 4 Schritten

  1. Google Sheet erstellen (z. B. „Finanz-Tracker“).
  2. Im Sheet: Erweiterungen → Apps Script öffnen und den Inhalt von apps-script/Code.gs einfügen. Das TOKEN oben im Code ändern.
  3. Bereitstellen → Neue Bereitstellung → Web-App: Ausführen als Ich, Zugriff Jeder. URL kopieren.
  4. URL und Token hier oben eintragen und „Speichern & verbinden“ klicken. Das Script legt die Tabellenblätter automatisch an.
